# Admin Dashboard

This is a **React + Vite project** for an admin dashboard.

## Features
- Responsive design
- Charts and cards components
- Weather and Todo List features
- Sidebar and Header components

## How to run locally
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:5173` in your browser

## Technologies
- React
- Vite
- JavaScript, HTML, CSS
